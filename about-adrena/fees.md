# Fees 🧀

> ⚠️ Full fee table to be populated. Run the local scraper script for complete content.

Adrena charges fees on:

- **Open position** — Flat fee on position size
- **Close position** — Flat fee on position size
- **Borrow fee** — Ongoing fee while a position is open (based on utilization)
- **Swap fees** — For adding/removing liquidity via ALP

All fees are distributed to ADX stakers and ALP holders.

# Governance Shadow Token 👤

> ⚠️ Page content to be fetched and populated. Run the local scraper script for full content.

The governance shadow token mechanism ensures that locked/vesting tokens still provide voting power, enabling the team and early contributors to guide the protocol during the initial build phase while remaining fully transparent and on-chain.
